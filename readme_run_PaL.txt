Run PaL in k.LAB software:
1. Import the project to the k.Modeler: https://bitbucket.org/integratedmodelling/im.ecology.grassland.livestock/src/master/
2. Select the spatio-temporal context in k.Explorer
3. Observe the model by typing the concept used to annotate the model in the knowledge search bar of the k.Explorer interface or drag-and-drop in k.Modeler

More information to get started with k.LAB:
https://aries.integratedmodelling.org/
